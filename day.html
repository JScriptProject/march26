My $20K Goal Modes - March 2026

Goal: Earn $20,000 by March 2026 via freelancing & blog monetization.

Survival Mode (6 Hours/Day)

Mindset: "This is my baseline—I must do this to survive."

Daily Breakdown

  • Learning React (1.5 hrs): 30 mins tutorial, 60 mins coding.
  • Freelancing (2 hrs): 30 mins bidding (5-10 jobs), 90 mins gigs/portfolio.
  • Blogging (1.5 hrs): 45 mins writing, 45 mins research/web story.
  • Twitter (1 hr): 30 mins posting (3-5 tweets), 30 mins engaging (5-10 accounts).

Weekly Targets: 1-2 React projects/month, 1-2 gigs/week, 2 posts + 1 web story/week, 20-50 followers/week.

Beast Mode (8 Hours/Day)

Mindset: "I’m stretching to dominate—faster growth."

Daily Breakdown

  • Learning React (1.5 hrs): 30 mins tutorial, 60 mins coding (1 project/week).
  • Freelancing (3 hrs): 60 mins bidding (10-15 jobs), 120 mins gigs/portfolio.
  • Blogging (2 hrs): 60 mins writing (3 posts/week), 60 mins research/1-2 web stories.
  • Twitter (1.5 hrs): 45 mins posting (5-7 tweets), 45 mins engaging (10-15 accounts).

Weekly Targets: 1 React project/week, 2-3 gigs/week, 3 posts + 2 web stories/week, 50-100 followers/week.

Dream Mode (10 Hours/Day)

Mindset: "I’m all-in—this is my dream push."

Daily Breakdown

  • Learning React (2 hrs): 45 mins tutorial (advanced), 75 mins coding (complex projects).
  • Freelancing (4 hrs): 90 mins bidding (15-20 jobs), 150 mins gigs/templates.
  • Blogging (2.5 hrs): 90 mins writing (4 posts/week), 60 mins research/2-3 web stories.
  • Twitter (1.5 hrs): 45 mins posting (7-10 tweets), 45 mins engaging (15-20 accounts).

Weekly Targets: 1-2 React projects/week, 3-5 gigs/week, 4 posts + 3 web stories/week, 100-200 followers/week.

Discipline Tips

  • Survival: Stick to timers, no distractions.
  • Beast: 5-10 min breaks every 2 hrs, 30-min midday reset.
  • Dream: Split into 2 blocks (e.g., 5+5 hrs), nap if needed.
  • General: Track tasks (Notion/Trello), sleep 7-8 hrs, exercise 30 mins.
